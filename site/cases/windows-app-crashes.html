Troubleshooting case

Find repeated Windows application crashes

Windows Diagnostics Toolkit reads recent Application Error, Windows Error Reporting, Application Hang, and BugCheck metadata. It also lists recent dump-file names, timestamps, sizes, and sources without opening dump contents.

pwsh -NoProfile -File .\Invoke-WindowsDiagnostics.ps1 -Crashes -PrivacyMode -ExportMarkdown

What the report can show

  • Repeated failures from the same provider or application
  • Recent system BugCheck events
  • Fresh user or system crash dumps
  • An aggregated WARN or ERROR finding before the detailed events

The tool does not fix applications, delete dumps, or change Windows Error Reporting settings.
