Troubleshooting case

Windows Firewall profiles are disabled

Collect a security posture snapshot without enabling, disabling, or reconfiguring protection.

pwsh -NoProfile -File .\Invoke-WindowsDiagnostics.ps1 -Security -PrivacyMode -ExportMarkdown

Context collected

Domain, Private, and Public firewall profiles plus Defender, Secure Boot, TPM, and BitLocker status.

Read-only boundary

The toolkit reports state only. It does not change firewall rules or security policy.

Interpretation

A disabled profile is evidence to investigate, not an automatic instruction to change the system. Confirm whether another security product or managed policy is responsible before remediation.
